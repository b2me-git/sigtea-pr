# Extensões VS Code Recomendadas

## 🔧 Essenciais

### 1. **ESLint**
- **ID**: `dbaeumer.vscode-eslint`
- **Descrição**: Integração com ESLint para linting em tempo real
- **Benefício**: Detecta erros e padrões de código imediatamente

### 2. **Prettier - Code Formatter**
- **ID**: `esbenp.prettier-vscode`
- **Descrição**: Formatação automática de código
- **Benefício**: Mantém codificação consistente sem configuração manual

### 3. **TypeScript Vue Plugin**
- **ID**: `ms-vscode.vscode-typescript-next`
- **Descrição**: Suporte avançado a TypeScript
- **Benefício**: Autocomplete, refatoração e type checking melhorados

## 🚀 Desenvolvimento Mobile

### 4. **React Native Tools**
- **ID**: `xabikos.ReactNativeTools`
- **Descrição**: Debug e complementos para React Native
- **Benefício**: Debugging, console logging, emulador integration

### 5. **Expo**
- **ID**: `byCedric.vscode-expo` (ou buscar por "Expo" na marketplace)
- **Descrição**: Suporte para Expo no VS Code
- **Benefício**: Integração com Expo CLI e metadata

### 6. **Simple React Snippets**
- **ID**: `burkeholland.simple-react-snippets`
- **Descrição**: Snippets para componentes React
- **Benefício**: Acelera criação de componentes

### 7. **ES7+ React/Redux/React-Native snippets**
- **ID**: `dsznajder.es7-react-js-snippets`
- **Descrição**: Snippets completos para ecossistema React
- **Benefício**: Autocompletar código comum

## 🎨 Styling & UI

### 8. **Tailwind CSS IntelliSense**
- **ID**: `bradlc.vscode-tailwindcss`
- **Descrição**: Autocomplete e preview para Tailwind CSS
- **Benefício**: IntelliSense para estilos (útil se usar Nativewind)

### 9. **Material Icon Theme**
- **ID**: `PKief.material-icon-theme`
- **Descrição**: Ícones de pasta e arquivo mais legíveis
- **Benefício**: Melhor visualização no Explorer

### 10. **Vscode-icons**
- **ID**: `vscode-icons-team.vscode-icons`
- **Descrição**: Ícones adicionais para VS Code
- **Benefício**: Melhor organização visual

## 🧪 Testes & Debug

### 11. **Jest**
- **ID**: `orta.vscode-jest`
- **Descrição**: Integração com Jest para testes
- **Benefício**: Execução e debugging de testes

### 12. **REST Client**
- **ID**: `humao.rest-client`
- **Descrição**: Cliente HTTP diretamente no editor
- **Benefício**: Teste APIs sem Postman

## 📝 Produtividade

### 13. **GitLens**
- **ID**: `eamodio.gitlens`
- **Descrição**: Git supercarregado para VS Code
- **Benefício**: Visualizar histórico, blame, commits

### 14. **TODO Highlight**
- **ID**: `wayou.vscode-todo-highlight`
- **Descrição**: Destaca TODO, FIXME, etc.
- **Benefício**: Acompanhamento de tarefas no código

### 15. **Better Comments**
- **ID**: `aaron-bond.better-comments`
- **Descrição**: Comentários coloridos e destacados
- **Benefício**: Melhor organização de notas

### 16. **Bookmarks**
- **ID**: `alefragnani.bookmarks`
- **Descrição**: Marca linhas importantes para retornar depois
- **Benefício**: Navegação rápida entre pontos importantes

### 17. **Indent Rainbow**
- **ID**: `oderwat.indent-rainbow`
- **Descrição**: Indentação com cores alternadas
- **Benefício**: Identifica níveis de indentação facilmente

## 🔐 Segurança & Env

### 18. **.env**
- **ID**: `mikestead.dotenv`
- **Descrição**: Syntax highlight para arquivos .env
- **Benefício**: Melhor visualização de variáveis de ambiente

### 19. **Error Lens**
- **ID**: `usernamehw.errorlens`
- **Descrição**: Exibe erros inline no editor
- **Benefício**: Detecta problemas sem usar o painel de problems

## 🔧 Configuração em Massa

Para instalar todas as extensões recomendadas de uma vez:

```bash
code --install-extension dbaeumer.vscode-eslint
code --install-extension esbenp.prettier-vscode
code --install-extension ms-vscode.vscode-typescript-next
code --install-extension dsznajder.es7-react-js-snippets
code --install-extension burkeholland.simple-react-snippets
code --install-extension xabikos.ReactNativeTools
code --install-extension bradlc.vscode-tailwindcss
code --install-extension orta.vscode-jest
code --install-extension eamodio.gitlens
code --install-extension wayou.vscode-todo-highlight
code --install-extension aaron-bond.better-comments
code --install-extension alefragnani.bookmarks
code --install-extension oderwat.indent-rainbow
code --install-extension mikestead.dotenv
code --install-extension PKief.material-icon-theme
code --install-extension vscode-icons-team.vscode-icons
code --install-extension usernamehw.errorlens
```

Ou via VS Code:
1. Abra Extensions (Ctrl+Shift+X)
2. Clique no botão "..." → "Install from .vscode/extensions.json"
